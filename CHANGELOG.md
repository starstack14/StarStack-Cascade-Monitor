# Changelog

Все заметные изменения StarStack Cascade Monitor фиксируются в этом файле. Формат основан на Keep a Changelog, версии следуют Semantic Versioning.

## [2.7.0] — 2026-07-18

### Added

- Редактирование тестовых URL Direct/Moscow/Germany прямо в окне настроек.
- Профили `home`, `demo` и `diagnostic` в меню приложения.
- Недельный текстовый отчёт с uptime, простоями, трафиком, задержкой и количеством событий.
- Журнал пользовательских действий.
- Контроль срока TLS-сертификата web-панели и предупреждение за 14 дней до окончания.
- Звуковое уведомление о новом критическом сбое.
- Восстановление конфигурации NX31 из защищённого backup с двойным подтверждением и проверкой архива.

### Security

- Восстановление требует ввода фразы `RESTORE NX31` и не выполняется без явного выбора backup.

## [2.6.0] — 2026-07-18

### Added

- Установка обновления в один клик из окна GitHub Release.
- Проверка размера и PE-заголовка скачанного EXE перед заменой текущей версии.
- Автоматический перезапуск приложения после успешной замены EXE.

## [2.5.0] — 2026-07-18

### Added

- Проверка доступности сайтов через настроенные правила Direct, Moscow и Germany.
- Карточка результатов с HTTP-статусом, задержкой или текстом ошибки.
- Режим демонстрации с горячей клавишей `Ctrl+Shift+H`.
- GitHub Actions workflow для публикации GitHub Release с готовым Windows EXE по тегу `v*`.

### Security

- Режим демонстрации скрывает IP-адреса, имена пользователей и устройств, а также трафик в desktop и web-панели.

## [2.4.0] — 2026-07-18

### Added

- Автоблокировка раскрытых IP-адресов через настраиваемые 30–60 секунд.
- Тайм-аут web-панели после настраиваемого периода без действий пользователя.
- Автоматическое определение первопричины сбоя: NX31, DNS, Moscow, Germany или Remnawave.
- Отдельный компактный экран для второго монитора с живым состоянием каскада.
- Фоновая проверка GitHub Releases, окно changelog и кнопки скачивания EXE / открытия релиза.

### Changed

- Автоматические запросы статуса web-панели больше не продлевают пользовательскую сессию.

## [2.3.0] — 2026-07-16

### Added

- Режим приватности для IP-адресов с сохранением состояния между запусками.
- Скрытие и отображение IP нажатием на адрес или кнопку `IP` в нижней панели.
- Аналогичное скрытие IP в мобильной web-панели с сохранением настройки в браузере.

### Security

- Удалён парольный fallback для доступа приложения к NX31: используется только SSH-ключ.
- Старое поле `router_password_dpapi` автоматически удаляется из локальной конфигурации.
- IP маскируются в маршрутах, WAN, активных подключениях, домашних устройствах и DNS/IPv6-отчёте.

## [2.2.0] — 2026-07-16

### Added

- Центр уведомлений с цветовой категоризацией и счётчиком непрочитанных событий.
- Таймлайн инцидентов с активными сбоями, временем восстановления и длительностью простоя.
- Хранение истории инцидентов за последние 30 дней в локальной SQLite-базе.
- Расширенные интерактивные графики RAM, задержки и нагрузки за 1 час, 24 часа и 7 дней.
- Быстрые действия на карточке NX31: график, проверка IP, DNS/IPv6 и перезапуск sing-box.
- Быстрые действия на карточках нод: график, тест скорости и обновление данных.
- Переход к расширенному графику нажатием на мини-график карточки.

### Security

- Генерируемый Caddyfile с пользовательским доменом перенесён в игнорируемую runtime-папку `logs`.

## [2.1.0] — 2026-07-16

### Changed

- Полностью обновлён интерфейс рабочего стола в стиле Neon Glass v3.
- Маршрут NX31 → Moscow → Germany теперь показан как отдельная живая цепочка.
- IP-адреса, состояние роутера и показатели нод разбиты на компактные информационные блоки.
- Карточки Moscow и Germany получили более ясные статусы, метрики и увеличенные графики RAM.
- Список подключений получил значки платформ и более читаемую компоновку устройств.
- Высота окна зафиксирована, длинное содержимое доступно через прокрутку колёсиком.
- Мобильная web-панель приведена к общей цветовой палитре desktop-приложения.

### Fixed

- Длинный список пользователей больше не растягивает виджет за границы рабочего стола.

## [2.0.0] — 2026-07-16

### Added

- Адаптивная мобильная web-панель для просмотра состояния каскада с телефона.
- Встроенный backend на `127.0.0.1:8765` и HTTPS reverse proxy Caddy.
- Автоматический сертификат Let’s Encrypt для настраиваемого домена.
- Отдельная web-авторизация с DPAPI-паролем, защищённой cookie-сессией и ограничением попыток входа.
- Read-only JSON API со статусом NX31, маршрутов, нод, пользователей и домашних устройств.
- Поддержка публикации через закреплённый LAN-IP и перенаправление TCP 80/443 на OpenWrt.

### Security

- Добавлены CSP, HSTS, `X-Frame-Options`, `X-Content-Type-Options` и `Referrer-Policy`.
- Backend привязан только к loopback-интерфейсу и не публикуется напрямую.
- Web-сессии хранятся только в памяти и сбрасываются при завершении приложения.

## [1.4.0] — 2026-07-16

### Added

- База известных домашних устройств и уведомления о новых MAC-адресах.
- Пользовательские имена и метки доверенных устройств.
- История подключения и отключения каждого устройства.
- Графики скорости за 24 часа и 7 дней.
- Контекстное меню для блокировки/разблокировки устройства на NX31.
- Просмотр полного списка известных устройств.

### Security

- Блокировка реального устройства требует подтверждения.
- Запрещена блокировка компьютера, на котором работает монитор.

## [1.3.0] — 2026-07-16

### Added

- `nlbwmon` на NX31 для учёта трафика по MAC-адресам.
- Текущая скорость, расход за день и текущий месяц для домашних устройств.
- Еженедельные автоматические резервные копии с ротацией последних 5 архивов.
- Диагностический снимок перед watchdog-перезапуском sing-box.
- Проверка DNS- и IPv6-утечек.
- Журнал событий и защита от второй копии приложения.

### Changed

- Watchdog ограничен тремя последовательными сбоями, паузой 10 минут и двумя попытками в час.
- Резервные копии защищаются Windows ACL.

## [1.2.0] — 2026-07-16

### Added

- Обновлённый полупрозрачный neon-интерфейс под рабочий стол.
- Флаги и короткие названия Moscow/Germany.
- Системный трей и скрытие окна вместо завершения по крестику.
- Контекстный тест скорости нод.
- Схема маршрутов и отображение Direct/Moscow/Germany IP.
- SSH-управление NX31 отдельным Ed25519-ключом.
- Просмотр журнала и ручной перезапуск sing-box.
- Подтверждаемая перезагрузка роутера.

## [1.1.0] — 2026-07-15

### Added

- Мониторинг Remnawave Nodes, Users и HWID Devices.
- Задержка, Load, RAM, трафик и число пользователей нод.
- Мониторинг OpenWrt через LuCI/ubus и SSH.
- SQLite-история и графики RAM.
- DPAPI-шифрование локальных секретов.
- Автозапуск Windows и сборка PyInstaller в один EXE.

## [1.0.0] — 2026-07-15

### Added

- Первая рабочая версия StarStack Cascade Monitor.
